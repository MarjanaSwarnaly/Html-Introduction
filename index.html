==>What is HTML

* HTML stands for Hyper Text Markup Language
* HTML uses for creating a web page structure
* Consists of elements to display contents

==>History of HTML

* HTML was created by Sir Tim Berners-Lee in 1991
* was published in 1995 * While publishing it came with HTML1 version
* In 1995 HTML2.0 comes, then HTML 3.0, then comes HTML 4.0 and now we are using HTML 5.

==>HTML Template

* The "DOCTYPE" declaration defines this document to be HTML5
* The "html" element is the root element of an HTML page
* The "head" element contains meta information about the document
* The "title" element specifies a title for the document
* The "body" element contains the visible page content
